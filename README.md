# nextjs-cnode

> A cnodejs client using next.js 


## Setup

```bash
npm install

npm run dev

```



## Build


```bash
npm run build

```


## Deploy

```
pm2 start

```


- https://github.com/zeit/next.js/wiki/Deployment-on-Nginx's-reverse-proxy
